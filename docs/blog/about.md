# vuepress使用感受
* 我感觉在vuepress在没有牛逼主题的加持下,只能作为接口文档分享,不适合做博客站点。
* 最好有个主题开箱即用,有标签云,有评论,有时间线,有阅读和字数统计,赞赏，友链功能
* vuepress的首页很好看,很简洁,真香。
* 目前写做博客站点还是用Hexo吧,推荐一个主题:hexo-theme-huweihuang 真香
